# Embedded C Fundamentals

## Objective
To build a strong foundation in Embedded C programming by understanding low-level concepts used in firmware development.

## Concepts Covered
- Bitwise operations
- Memory layout (stack vs heap)
- Basic embedded logic using C

## Tools & Environment
- Language: C
- Platform: Linux / Termux
- Compiler: GCC

## Learning Outcome
This repository helps in understanding core Embedded C concepts required for embedded and firmware engineering roles.
